The StrawPot web dashboard provides a browser-based interface for managing
projects, launching sessions, browsing resources, and scheduling automated
tasks — all without touching the command line.

```bash
strawpot gui
```

Opens [http://localhost:8741](http://localhost:8741) in your browser.

## Install

The GUI is included with StrawPot:

```bash
pip install strawpot
```

## How It Works

The dashboard runs a FastAPI server that:

1. Stores project registrations and session metadata in a local SQLite database (`~/.strawpot/gui.db`)
2. Launches sessions as headless subprocesses via `strawpot start --headless --task "..."`
3. Reads trace files (`.strawpot/sessions/*/trace.jsonl`) for session detail views
4. Runs a background cron scheduler that fires scheduled tasks automatically

All data stays local. No external services or accounts are required.

## Pages

| Page | Path | Description |
|------|------|-------------|
| Dashboard | `/` | Running sessions, recent sessions, quick launch |
| Projects | `/projects` | List and register projects |
| Project Detail | `/projects/:id` | Sessions, resources, files, and config for a project |
| Session Detail | `/projects/:id/sessions/:runId` | Agent tree, trace events, logs |
| Schedules | `/schedules` | Manage cron-based scheduled tasks |
| Resources | `/resources/:kind` | Browse installed roles, skills, agents, memories |
| Settings | `/settings` | Global configuration |
